publish/3

publish(Group::group(), Tag::tag(), Data::term()) -> ok | {error, term()}

Publishes Data to Group and broadcasts it to all subscribers. All messages have to be tagged, allowing subscribers to easily pattern-match on the messages they receive.

statistics/1

statistics(Group::group()) -> {ok, [{atom(), term()}]} | {error, term()}

Returns a property list with statistics about a given group.

Currently the following keys are returned:
group: The name of the group
subscriber_count: Current number of subscribers
message_count: Number of unique messages this process (group) sent
message_queue: Number of messages that are currently queued up (actual Erlang message queue)
