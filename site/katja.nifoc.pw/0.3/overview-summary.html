Version: 0.3

Authors: Daniel Kempkens (daniel@kempkens.io).

Katja is a simple Riemann client written in Erlang. The source code can be found on GitHub.

The public API of Katja can be found in the katja module.

The default configuration looks like this:

[
  {katja, [
    {host, "127.0.0.1"},
    {port, 5555},
    {pool, []}
  ]}
].
